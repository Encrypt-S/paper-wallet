What you need to know

None of the generated keys are ever broadcast over the internet. It is recommended that run it locally on your computer instead of generating the keys directly on the site.

For the extra paranoid; you can run the generator and print the wallets using a secure offline environment and correctly wipe the environment and printer memory after use.

Download the Generator

You can generate up to 4 which will fit onto 1 piece of portrait A4 paper.

Encrypting the private keys with a BIP38 password is recommended, but optional. Encrypting can take some time to complete, so please be patient. The password you set will be used for all generated wallets. This password can not be recovered, so do not lose it!

It is recommended that you test decrypting the private key here before sending coins to it to check nothing went wrong during encrytion.
